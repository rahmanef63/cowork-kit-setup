# projects/

Your generated automation projects land here — one folder per field.

You don't create these by hand. Install the `cowork-automation-generator` skill,
then in Cowork just say what you do (e.g. *"I'm an accountant, set up Cowork for
me"*). The skill interviews you and scaffolds `projects/<your-field>/`, ready to
use in Cowork.

Generated projects are gitignored by default (they're yours, not part of the
template). Keep one if you want by force-adding it.
